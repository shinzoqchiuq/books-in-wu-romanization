# 赞美诗

《赞美诗》出版于 1860 年，收录了当时派驻宁波的基督教传教士所作的 166 首赞美诗与 9 首颂赞歌。

书影来自 [普林斯顿大学图书馆](https://catalog.princeton.edu/catalog/9930372423506421)。

由于原书影印版缺页，缺少第 34、35、48、49、162、163 首赞美诗以及第 36 首与 47 首缺少部分诗句。这里根据 1874 年版做了补全。1874 年版书影来自 [牛津大学博德利图书馆](https://east-asian.bodleian.ox.ac.uk/catalog/volume_1808)。
