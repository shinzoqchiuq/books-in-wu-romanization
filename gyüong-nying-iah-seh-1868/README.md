# 穷人约瑟

书影来自 [牛津大学博德利图书馆](https://east-asian.bodleian.ox.ac.uk/catalog/volume_808)。

一则小故事，出版于 1868 年。未属作者名，据博德利图书馆信息，作者为中国内地会传教士江郎笔（George Crombie）。
