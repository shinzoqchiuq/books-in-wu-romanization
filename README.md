# 罗马字宁波话文献整理转写

[![已转写字数](https://img.shields.io/badge/已转写-26.51_万字-blue?style=flat-square)](./字.tsv)

十九世纪五十年代至二十世纪三十年代，派驻宁波的基督教传教士使用一套统一的罗马字方案，出版了众多宁波方言（宁波土话/宁波土白）书籍。这些出版物多数只有书影，未被整理为文本格式。本仓库将这些用罗马字拼写的宁波方言出版物整理为文本格式，并将内容转写为方言正字，方便研究、学习。

本仓库目前整理并转写了《地理书·卷四》《创世记》《新约书（部分）》《赞美真神诗》《赞美诗》《穷人约瑟》《三本小文书》《小显理等其个底下人蒲齐》。并会不定期更新。

本仓库转写所依据的书影来自互联网，具体链接请见各子文件夹的 README。所有书籍的作者或译者均已去世超过 50 年，原始文本已进入公有领域。由于纸质书籍的拍摄本或扫描本可能存在再分发限制或影像著作权，因此本仓库仅提供文本格式的原文及其汉字转写，不提供 PDF 格式的书影。如需原书书影，请至原始链接获取。本仓库中的汉字转写文本依据 CC0 1.0 协议放弃全部著作权及相关权利，直接进入公有领域。

## 宁波话传教士罗马字简介

本仓库所谓宁波话传教士罗马字，指的是美北长老会传教士丁韪良于 1851 年创制的一套用来拼写/书写当时宁波方言的拼音文字方案。该方案经派驻宁波的传教士团体共同完善，推测于 1852 年定型。后被用于诸多宁波方言书籍的编写。该方案只拼写声、韵母，不标声调。拼写以音节为单位，双音节或多音节词语用「-」连接音节。如「鄞县」拼作 Nying-yün。该套拼音系统反映了十九世纪中叶宁波府城方言的语音面貌，与今天的宁波方言有诸多区别。
