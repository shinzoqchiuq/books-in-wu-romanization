# 四则故事

书影来自 [牛津大学博德利图书馆](https://east-asian.bodleian.ox.ac.uk/catalog/volume_831)。

四则小故事：〈路孝子〉〈父子算账〉〈一杯酒〉〈虽拉等喊哪〉，出版于 1869 年。封面手书「Fra（标签遮挡）other stories. Translated into the Ningpo Dialect by D. B. McCartee M. D.」表明这四则故事的译者为美北长老会传教士麦嘉缔（Divie Bethune McCartee）。

〈路孝子〉与〈一杯酒〉这两则故事曾在 1852 年单独出版过，书影可在哈佛大学燕京图书馆找到：《[路孝子](https://id.lib.harvard.edu/alma/990081478050203941/catalog)》《[一杯酒](https://id.lib.harvard.edu/alma/990081478000203941/catalog)》。前人对 1852 年出版的这两本书有做过汉字转写<sup>\[1–3\]</sup>。燕京图书馆网页上将《一杯酒》的作者标为英国圣公会传教士高富（Frederick Foster Gough），但书影各处均未见属名。可能是图书馆信息有误，这四则故事的译者应当都是麦嘉缔。

与 1852 年版的《路孝子》《一杯酒》相比，故事文本差异不大。本仓库以注释的形式标出了两个版本间除标点和分词之外的所有差异。

〈路孝子〉来源于美国近代乡村传说 Stories of Frank Lucas<sup>\[1\]</sup>。书影封面被标签遮挡的文字推测即为「Frank Lucas」。

〈一杯酒〉来源于小亚细亚地区的希腊人故事 A cup of wine<sup>\[1\]</sup>。

〈父子算账〉和〈虽拉等喊哪〉这两则故事没有找到可靠的来源，可能是当时新教传教士当中流行的宗教寓言故事。
