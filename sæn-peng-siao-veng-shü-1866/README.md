# 三本小文书

书影来自 [牛津大学博德利图书馆](https://east-asian.bodleian.ox.ac.uk/catalog/volume_2234)。

三则小故事，〈亮塔个小人〉〈孤哀子碰着朋友〉〈懊悔说话欠早〉，出版于 1866 年。据扉页信息，「箇三本小文书是慕师母所造个」，即作者为慕师母，可能是英国圣公会传教士慕稼谷（George Evans Moule）的夫人或其弟慕雅德（Arthur Evans Moule）的夫人。
